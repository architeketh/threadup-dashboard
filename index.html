📤 How to Share & Track Sales

  1. Upload your data files (sold + available items)
  2. Click "✓ Mark as Updated" to start tracking new sales
  3. Click "📤 Export for GitHub" - downloads sold.json, available.json, and locked-dates.json
  4. In your repo, go to data/ folder on GitHub
  5. Click "Upload files" and add ALL JSON files (including locked-dates.json)
  6. Commit changes
  7. Share: https://architeketh.github.io/threadup-dashboard/
  8. Next time you upload, you'll see 🔥 New Sales since last update!

🔒 Date Protection System

Manual edits are PERMANENTLY PROTECTED and synced via GitHub:
• When you edit a date in the Sold Items table, it's locked forever
• Click "💾 Save Changes to GitHub" to sync locked dates
• Locked dates automatically load from GitHub on any device
• Your edits work across ALL browsers and devices!

Serial Number Formats by Era

Pre-1994 · Vintage
No number present
Vintage bags have only the creed text — no style number. Authenticity relies on leather quality, hardware, and craftsmanship.
1994 – 2006 · Classic
K8P-9870  ·  E05Q-9950
Letter + 1–2 digits + letter, dash, 4-digit style number. Letters before the dash encode factory, year, and production month.
2006 – 2020 · Modern
B1980-38124  ·  J0972-F13084
Letter + 4 digits, dash, 5-digit style number. An optional letter prefix on the style number (F = Outlet, P = Pilot) is a recognized Coach convention.
2020 – Present · Current
H2081-C0769  ·  L2023-6019  ·  H2039-601
Several new formats: shorter 3–4 digit style numbers, "C"-prefix numbers, two-letter codes. Bags may also have a QR code or NFC chip.
⚠ Common Signs of a Counterfeit
✗Number printed on paper or sewn-in tag instead of stamped into leather
✗Uneven, wobbly, or faded stamp impressions
✗Creed patch has spelling errors or unusual phrasing
✗Style number does not match the bag's era or actual model
✗Hardware feels lightweight, tarnishes quickly, or has poorly cast "Coach" lettering
✗Lining is thin, has loose threads, or a strong chemical smell
